# USS Digital Kit（進行中）

**USS Digital Kit** は、Universidad San Sebastián における設計・開発体験を底上げするためのプロジェクトです。**Vanilla JavaScript 版** と **React 版** の 2 つの補完的ライブラリを整備し、従来型システムとモダン開発の両方をカバーします。

このキットは既存デザインシステムの進化形として、アクセシビリティ・モジュール性・開発者体験に関する最新の実践を取り入れています。

## 主要特性

- **デュアル実装**（Vanilla JS / React）による高い柔軟性
- 設計段階から ARIA を含む **高度なアクセシビリティ**
- 例とガイドを備えた **包括的ドキュメント**
- 機関要件に対応する **テーマシステム**
- 必要な部品だけ読み込める **モジュールアーキテクチャ**

## 使用技術

- 共通基盤: **JavaScript ES2023** / セマンティック HTML
- スタイル: **CSS/SCSS**（BEM + Custom Properties）
- バンドル: **Rollup**（最適化された tree-shaking）
- UI 実装: **React 18+**
- 型安全: **TypeScript**（React 版）

## 担当内容

**Vanilla JS 版** と **React 版** の両方で、以下を担当しています。

- コア／応用コンポーネントの実装
- アクセシビリティパターンと ARIA ロール設計
- **SCSS + CSS Custom Properties** による拡張可能なスタイル基盤
- 公式デザインシステムのトークンを実装へ直接反映
- 再利用しやすいコンポーネントドキュメント整備
