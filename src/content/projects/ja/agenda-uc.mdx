# Agenda UC

**Agenda UC** は、大学内の学術・文化・運営系アクティビティを公開・検索するための統合プラットフォームです。学生・教職員が必要なイベントを見つけ、整理しやすくすることを目的にしています。

プラットフォームは **Strapi バックエンド** と **React/Next.js フロントエンド** で構築し、大学全体のデジタル体験と整合性を保つため **UC Digital Kit** のコンポーネントを活用しています。

## 担当内容

- **データモデル設計（Strapi API）**  
  複数日程・繰り返し開催・イベント間リレーションに対応するデータ構造の定義と設計に参加しました。単発イベントから複数セッションの会議まで扱える柔軟な API を実現しました。

- **パフォーマンス最適化**  
  クエリとリレーションを最適化し、大量データやフィルタ利用時でも整合性を保ちながら高速に表示できるようにしました。

- **フロントエンド開発（React/Next.js）**  
  **UC Digital Kit** を土台に Agenda の UI を実装しました。必要に応じてガイドライン準拠のカスタムコンポーネントも作成し、一貫した体験を維持しました。

- **アクセシビリティの標準化**  
  **WCAG 2.1 AA** に沿って、フィルタ・イベント一覧・ナビゲーションがキーボードやスクリーンリーダーで利用可能になるよう設計しました。アクセシビリティは初期段階から要件として組み込んでいます。

## 使用技術

- イベント API: **Strapi (Node.js)**
- ダイナミックなフロントエンド: **React + Next.js**
- コンポーネント基盤: **UC Digital Kit**
- 見た目の調整: **CSS/SCSS**

## インパクト

Agenda UC により、大学コミュニティのイベント探索と参加導線は大きく改善されました。柔軟なアーキテクチャとアクセシブルな UI によって、多くの利用者が UC の他サービスと整合した形で情報へアクセスできます。
