# UC Digital Kit

**UC Digital Kit** は、チリ・カトリック大学のデジタルエコシステムを支える基盤です。**Vanilla JavaScript 製のコンポーネントライブラリ**として、100 以上の機関サイトで視覚的一貫性、アクセシビリティ、パフォーマンスを保証します。

Vanilla JS を選んだ理由は明確で、モダンフレームワークの有無に関係なく **どのサイトでも動作すること** が要件だったためです。これにより、新規アプリケーションでも既存のレガシー環境でも同じコンポーネントを摩擦なく導入できます。

## 担当内容

当初はアクセシビリティとユーザビリティの改善から参加し、その後はより広範な技術設計と実装を担当しました。

- **dynamic import による読み込み最適化**  
  従来の一括読み込みを、DOM を解析して必要コンポーネントだけを配信する方式に置き換えました。各サイトが必要最小限のみ取得するため、全体の初期表示性能が向上しました。

- **アクセシビリティとセマンティクスの改善**  
  50 以上のコンポーネントで `background-image` を `<img>` + `alt` に置換。スクリーンリーダー対応を強化し、構造化の品質や検索エンジンからの解釈も改善しました。

- **配布と互換性維持**  
  **CDN** と **NPM** の両経路で配布し、ビルドあり／なし双方の環境をサポート。運用中プロジェクトを壊さない後方互換性を重視して更新を行っています。

- **包括的アクセシビリティ監査**  
  現在は **WCAG 2.1 AA** 基準でコンポーネント全体の監査を主導し、改善が **UC Portal** や **Agenda UC** などの依存プロジェクトへ即時に波及する体制を整えています。

## 使用技術

- 汎用互換性: **Vanilla JavaScript**
- 再利用可能なスタイル: **CSS/SCSS**
- 配布: **CDN / NPM**

## インパクト

UC Digital Kit は単なるライブラリではなく、大学 Web 全体の **共通インフラ** です。性能・アクセシビリティ・操作性の改善は、同一基盤を使う多数のサイトへ同時に反映され、学生・教職員すべての体験向上に直結します。
