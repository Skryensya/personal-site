# Hostal Micelio

**Hostal Micelio** は、ビジャリカにあるホステル向けに制作したセールスファネル型ランディングページです。目的は、必要情報を分かりやすく提示し、ユーザーを WhatsApp 経由の予約へ自然に導くことでした。

## プロジェクトの目的

体験設計の軸はシンプルさです。宿の魅力を短時間で伝え、信頼を生み、予約アクションまでの摩擦を最小化しました。コンパクトながら成果に直結する構成を重視しています。

## 担当内容

このプロジェクトでは、ビジュアルコンセプトから実装まで一貫して担当しました。

- 宿の個性を反映したカスタム UI デザイン
- 予約導線に集中させるシンプルなナビゲーション
- 施設紹介・客室・サービス・アクセス・CTA を明確に分離した情報設計
- 各デバイスでの快適な表示を意識したパフォーマンス最適化

## 使用技術

- 開発基盤: **React + Next.js**
- スタイリング: **Tailwind CSS**

## ページ導線

1. 予約を促すヒーローセクション
2. 宿の特徴とサービスを伝える情報ブロック
3. アクセス・連絡情報の提示
4. WhatsApp への継続的な CTA

## 学び

- クライアントのブランドに沿った自由度の高いデザイン判断
- 必要要素だけで行動を促すミニマル UX の実践
- コンバージョン重視の LP 制作における Next.js / Tailwind の活用
