## API Agenda UC

Sistema integral de gestión de eventos universitarios que centraliza toda la información de actividades académicas, culturales y administrativas de la Universidad Católica. Una solución completa que conecta estudiantes, profesores y personal administrativo con los eventos de su interés.

### Características Principales

- **Gestión completa de eventos** con categorización y filtros avanzados
- **API REST robusta** para integración con múltiples plataformas
- **Panel administrativo intuitivo** para organizadores de eventos
- **Notificaciones automáticas** para eventos de interés
- **Búsqueda y filtrado avanzado** por categorías, fechas y ubicación

### Tecnologías Utilizadas

- **Node.js con Strapi CMS** para backend y gestión de contenido
- **MySQL** como base de datos principal
- **React con Next.js** para el frontend interactivo
- **Kit Digital UC** para componentes de interfaz consistentes
- **API RESTful** para comunicación entre servicios

### Funcionalidades Clave

- Dashboard administrativo para gestión de eventos
- Sistema de categorías y etiquetas personalizables
- Integración con calendarios externos (Google Calendar, Outlook)
- Sistema de inscripción y lista de espera
- Reportes de asistencia y estadísticas de eventos

### Impacto

La plataforma ha mejorado significativamente la comunicación de eventos dentro de la universidad, aumentando la participación estudiantil y optimizando la gestión administrativa. Los organizadores ahora pueden crear, gestionar y promocionar eventos de manera más eficiente.
