## Portal UC

El sitio web principal y punto de entrada digital de la Universidad Católica de Chile. Este proyecto involucra el desarrollo continuo de funcionalidades, mantenimiento técnico y optimización enfocada en accesibilidad web, navegación intuitiva y posicionamiento SEO.

### Características Principales

- **Accesibilidad WCAG 2.1 AA** garantizada en todas las páginas
- **SEO optimizado** para máxima visibilidad en buscadores
- **Navegación intuitiva** diseñada para diferentes tipos de usuarios
- **Rendimiento optimizado** con tiempos de carga mínimos
- **Responsive design** para todos los dispositivos

### Tecnologías Utilizadas

- **PHP con ProcessWire CMS** para gestión flexible de contenido
- **Vue.js** para componentes interactivos y dinámicos
- **Kit Digital UC** para mantener consistencia visual
- **Optimización SEO avanzada** con structured data
- **Herramientas de accesibilidad** integradas

### Funcionalidades Desarrolladas

- Sistema de navegación adaptativo por audiencias
- Optimización de rendimiento y Core Web Vitals
- Implementación de structured data para mejores rich snippets
- Sistema de búsqueda avanzada con filtros inteligentes
- Integración con sistemas académicos y administrativos

### Impacto en la Institución

El portal sirve como la principal ventana digital de la universidad, recibiendo miles de visitantes diarios incluyendo estudiantes actuales, postulantes, académicos y público general. Las mejoras implementadas han resultado en:

- Mejor posicionamiento en motores de búsqueda
- Mayor accesibilidad para usuarios con discapacidades
- Reducción en tiempos de carga y mejora de la experiencia de usuario
- Aumento en la tasa de conversión de postulantes
