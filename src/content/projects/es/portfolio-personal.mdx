## Contexto

Este portfolio personal representa la culminación de mi experiencia en desarrollo web frontend, combinando tecnologías modernas con un diseño brutalmente minimalista que prioriza la funcionalidad sobre la decoración.

La filosofía del proyecto se basa en tres pilares fundamentales: **accesibilidad total**, **rendimiento óptimo** y **experiencia de usuario sin fricciones**.

## Tecnologías y Arquitectura

El sitio está construido con **Astro.js 5.12.3** como framework principal, aprovechando la generación estática de sitios para obtener tiempos de carga ultrarrápidos. React se utiliza únicamente para componentes interactivos específicos, siguiendo el patrón de "islas de hidratación" que minimiza el JavaScript del cliente.

**Tailwind CSS** proporciona el sistema de diseño con una paleta estrictamente limitada a dos colores que se invierten entre modo claro y oscuro, manteniendo siempre el máximo contraste para la accesibilidad.

## Sistema de Internacionalización

Implementé un sistema de i18n completamente funcional que soporta tres idiomas:
- Español (idioma por defecto, sin prefijo en URL)
- Inglés (prefijo `/en/`)
- Noruego (prefijo `/no/`)

El sistema incluye detección automática de idioma, rutas localizadas y un diccionario centralizado de traducciones que garantiza coherencia en toda la aplicación.

## Rendimiento y Optimización

El sitio alcanza puntuaciones perfectas en Lighthouse gracias a:
- Generación estática que elimina el tiempo de servidor
- Optimización automática de imágenes con formatos WebP
- Carga diferida de componentes React no críticos
- CSS crítico inlineado para el above-the-fold
- Bundle de JavaScript inferior a 150kb gzipped

## Accesibilidad y Usabilidad

Cada componente fue diseñado desde cero considerando las pautas WCAG:
- Navegación completa por teclado
- Soporte total para lectores de pantalla
- Ratios de contraste que superan AAA
- Textos alternativos descriptivos en todas las imágenes
- Foco visible y semántica HTML correcta
