## Kit Digital UC

Librería completa de componentes web para el ecosistema de sitios web de la Universidad Católica, utilizada en más de 100 sitios institucionales. Este proyecto representa el desarrollo de una solución integral para estandarizar y mejorar la experiencia digital de toda la universidad.

### Características Principales

- **Componentes accesibles** siguiendo estándares WCAG 2.1 AA
- **Diseño consistente** a través de todos los sitios de la UC
- **Fácil implementación** con documentación completa
- **Optimización de rendimiento** para carga rápida
- **Compatibilidad cross-browser** garantizada

### Tecnologías Utilizadas

- **JavaScript ES6+** para lógica de componentes
- **HTML semántico** para estructura accesible
- **CSS con SASS** para estilos modulares y mantenibles
- **Rollup** como bundler para optimización
- **Vue.js** para componentes interactivos
- **Nuxt.js** para implementación en proyectos SSR

### Impacto

Este proyecto ha transformado la presencia digital de la Universidad Católica, mejorando la consistencia visual y la experiencia de usuario en todo el ecosistema web institucional. La librería asegura que todos los sitios mantengan los estándares de accesibilidad y usabilidad más altos.

### Mi Rol

Como desarrollador principal, fui responsable del desarrollo completo de los componentes, implementación de estándares de accesibilidad, optimización de rendimiento y creación de la documentación técnica para otros desarrolladores.
