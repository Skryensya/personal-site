# Hostal Micelio

**Hostal Micelio** es una landing page creada como funnel de ventas para un hostal ubicado en Villarrica. El objetivo fue desarrollar un sitio sencillo pero efectivo, que guíe al usuario a reservar directamente a través de un canal de WhatsApp.

## Objetivo del proyecto

El enfoque principal fue construir una experiencia clara y directa: mostrar la información esencial del hostal, transmitir confianza y facilitar al máximo la acción de reservar. La página funciona como una vitrina digital simple pero eficiente, adaptada a las necesidades específicas del cliente.

## Contribución

En este proyecto trabajé de principio a fin, desde la concepción visual hasta la implementación en código. Mis aportes incluyeron:

- Diseño visual propio, tomando el control creativo para reflejar la identidad del hostal.
- Estructura de navegación simple, pensada como un embudo que lleva al usuario hacia la reserva.
- Construcción de secciones limpias y ordenadas: presentación del hostal, habitaciones, servicios, ubicación y un llamado a la acción claro.
- Optimización del rendimiento para asegurar una experiencia rápida y fluida en todo tipo de dispositivos.

## Tecnologías utilizadas

- **React + Next.js** como base de desarrollo.
- **Tailwind CSS** para la maquetación y estilos visuales.

## Funcionamiento

La landing sigue un recorrido claro:

1. Encabezado atractivo con llamado a reservar.
2. Secciones informativas con los servicios y características del hostal.
3. Información de ubicación y contacto.
4. Call-to-action persistente que lleva al usuario a concretar la reserva vía WhatsApp.

## Lecciones y aprendizajes

Este proyecto me permitió:

- Explorar un diseño completamente libre, tomando decisiones creativas basadas en la identidad del cliente.
- Aplicar principios de UX minimalista, mostrando solo lo necesario para guiar al usuario.
- Profundizar en el uso de Next.js y Tailwind CSS en la construcción de landings orientadas a conversión.
