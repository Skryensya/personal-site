## API Agenda UC

Omfattende universitetsarrangementsstyringssystem som sentraliserer all akademisk, kulturell og administrativ aktivitetsinformasjon ved Universidad Católica. En komplett løsning som forbinder studenter, professorer og administrativt personale med arrangementer av deres interesse.

### Hovedfunksjoner

- **Komplett arrangementsstyring** med kategorisering og avanserte filtre
- **Robust REST API** for flerdplatformsintegrasjon
- **Intuitivt administratorpanel** for arrangørene
- **Automatiske varsler** for arrangementer av interesse
- **Avansert søk og filtrering** etter kategorier, datoer og sted

### Teknologier Brukt

- **Node.js med Strapi CMS** for backend og innholdstyring
- **MySQL** som primær database
- **React med Next.js** for interaktiv frontend
- **Kit Digital UC** for konsistente grensesnittkomponenter
- **RESTful API** for tjenestekommunikasjon

### Hovedfunksjonaliteter

- Administrativt dashbord for arrangementsstyring
- Tilpassbart kategori- og merkelappssystem
- Ekstern kalenderintegrasjon (Google Calendar, Outlook)
- Registreringssystem og ventelister
- Deltakelsesrapporter og arrangementsstatistikk

### Påvirkning

Plattformen har betydelig forbedret arrangementskommunikasjon innenfor universitetet, økt studentdeltakelse og optimalisert administrativ styring. Arrangører kan nå opprette, administrere og promotere arrangementer mer effektivt.

### Min Rolle

Jeg utviklet den komplette frontend-applikasjonen og samarbeidet tett om API-datamodelldesignet. Arbeidet mitt inkluderte brukergrensesnittimplementering, API-integrasjon og optimalisering av sluttbrukeropplevelse.
