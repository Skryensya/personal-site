**Resumen del Proyecto:**  
**Barrancas**, nombre corto para "Plataforma Digital de Gestión Integral de Intervenciones Psicosociales en Salud Mental Escolar," es una plataforma de recolección de encuestas diseñada para asistir a profesores en tres comunas específicas de Santiago. Los profesores utilizan esta plataforma para llenar formularios que ayudan a identificar y apoyar a estudiantes en riesgo de deserción escolar y/o con problemas de salud mental. El objetivo de la plataforma es realizar análisis estadísticos de los datos recolectados para contribuir en este esfuerzo.

## Objetivos

1. Desarrollar una plataforma que permita a los profesores llenar encuestas para identificar estudiantes en riesgo.
2. Realizar análisis estadísticos de los datos recolectados para apoyar en la toma de decisiones.
3. Asegurar un flujo de trabajo eficiente y una interfaz fácil de usar para los profesores.

## Funcionalidades

1. **Recolección de Encuestas:**

- Los profesores pueden completar formularios en línea, que luego son procesados para identificar a estudiantes en riesgo.

2. **Análisis de Datos:**

- El backend realiza análisis estadísticos utilizando **Python** y **Pandas** para proporcionar reportes detallados sobre los estudiantes en riesgo.

3. **Desarrollo Integral:**

- La plataforma se ha desarrollado usando **Vue.js** para el frontend y **AdonisJS** para el backend, con **TailwindCSS** para el diseño visual y **MySQL** como base de datos.

## Stack Tecnológico

- Frontend:
  - **Vue.js** para la creación de una interfaz de usuario interactiva.
  - **TailwindCSS** para el diseño visual y la maquetación.
- Backend:
  - **AdonisJS** para la gestión del servidor y lógica de negocio.
  - **MySQL** como base de datos.
- Procesamiento de Datos:
  - **Python** (Pandas) para análisis y procesamiento de los datos.

## Resultados

El proyecto **Barrancas** ha facilitado la identificación y el apoyo de estudiantes en riesgo en las comunas seleccionadas. Como desarrollador principal del proyecto, he liderado el desarrollo de extremo a extremo del sitio, y actualmente estoy ajustando los últimos componentes para satisfacer los requisitos en constante evolución.
