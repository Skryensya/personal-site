**Resumen del Proyecto:**
Kit Digital UC es una librería de componentes creada para unificar y optimizar el ecosistema de sitios web de la Universidad Católica de Chile. Esta librería estandariza la identidad visual institucional, asegurando consistencia en todos los sitios web asociados.

## Objetivos

1.  Desarrollar una librería de componentes utilizada en la mayoría de los sitios web del ecosistema UC.
2.  Unificar la identidad visual de la universidad mediante componentes reutilizables.
3.  Mejorar la accesibilidad de los sitios web alineándose con los estándares WCAG 2.1 AA.

## Funcionalidades

1. **Componentes Reutilizables:**

- La librería ofrece una serie de componentes reutilizables que permiten construir sitios web consistentes con la imagen institucional de la UC.

2. **Documentación Completa:**

- El sitio de documentación desarrollado con Vue.js (Nuxt.js) facilita a los desarrolladores la implementación de los componentes del Kit Digital UC.

3. **Accesibilidad Mejorada:**

- Se ha trabajado en la accesibilidad de los componentes para cumplir con los requisitos de WCAG 2.1 AA, asegurando una mejor experiencia para todos los usuarios.

## Stack Tecnológico

- Librería de Componentes:
  - Vanilla JS para manipulación del DOM.
  - HTML y CSS para la estructura y estilo de los componentes.
  - Webpack para empaquetar y optimizar el código.
- Sitio de Documentación:
  - Vue.js (Nuxt.js) para generar una documentación clara y accesible.

## Resultados

El proyecto Kit Digital UC ha sido clave para estandarizar y mejorar la experiencia de usuario en los sitios web de la universidad. Su implementación ha permitido una integración más fluida de nuevas funcionalidades y una mejora significativa en la accesibilidad.
