**Resumen del Proyecto:**  
El proyecto **Kit Digital USS** tiene como objetivo crear una versátil librería de componentes para la Universidad San Sebastián (USS), basándose en la experiencia adquirida en el desarrollo del Kit Digital UC. Este kit completo incluirá una librería de componentes en JavaScript puro (vanilla JS), una versión basada en **React**, y un sitio de documentación para guiar a los usuarios. El objetivo principal es poner a disposición de todas las entidades de la universidad los componentes necesarios para construir sitios web con una identidad visual unificada y garantizar un nivel de usabilidad consistente en toda la institución.

## Objetivos

1. Desarrollar una librería de componentes reutilizables en JavaScript y React para la USS.
2. Crear un sitio de documentación que sirva como recurso esencial para los desarrolladores y diseñadores de la universidad.
3. Asegurar que los componentes desarrollados sigan una identidad visual unificada y accesible, fomentando la consistencia.

## Funcionalidades

1. **Librería de Componentes en JavaScript:**

- El **Kit base** incluye componentes construidos en **JavaScript** para la manipulación del DOM, **CSS** aplicando la metodología **BEM**, y **HTML**. Los componentes se empaquetan utilizando **Rollup**.

2. **Versión de React:**

- El **Kit React** aprovecha los estilos del **Kit base** y ofrece una implementación en **React.js** de los mismos componentes para una experiencia de desarrollo moderna.

3. **Sitio de Documentación:**

- El sitio de documentación se desarrolla utilizando **Next.js**, e integra el **Kit USS React** directamente desde **NPM** para proporcionar ejemplos en tiempo real y guías detalladas de uso.

## Stack Tecnológico

- **Kit base:**
  - **JavaScript** para la manipulación del DOM.
  - **CSS** aplicando la metodología **BEM** para mantener consistencia y mantenibilidad.
  - **HTML** para la estructura de los componentes.
  - **Rollup** para el empaquetado y optimización del código.
- **Kit React:**
  - **React.js** para la versión moderna de los componentes.
  - Estilos reutilizados del **Kit base**.
- **Sitio de Documentación:**
  - **Next.js** para el desarrollo del sitio de documentación.
  - **Kit USS React** integrado desde **NPM**.

## Resultados

El **Kit Digital USS** está en desarrollo y se espera que sirva como una herramienta fundamental para asegurar la coherencia visual y la accesibilidad en los sitios web de la Universidad San Sebastián. Mi contribución ha sido clave en el desarrollo de los requisitos del kit, creando un roadmap de verificación para asegurar un desarrollo más eficiente. También he promovido la accesibilidad y la adopción de la metodología **BEM** para mejorar la mantenibilidad y consistencia del código.
